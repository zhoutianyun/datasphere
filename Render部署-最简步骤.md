# Render 部署最简步骤

## 你要上传什么

把整个 `三机对抗-联机房间版` 文件夹放进一个 Git 仓库里，然后推到 GitHub。

关键文件已经配好：

- `render.yaml`
- `游戏文件/server.py`
- `游戏文件/requirements.txt`

## 最短部署流程

1. 打开 Render
2. 选择 `New +`
3. 选择 `Blueprint`
4. 连接你的 GitHub 仓库
5. 选中这个项目所在仓库
6. Render 会自动读到 `render.yaml`
7. 点击创建并等待部署完成

## 部署成功后

Render 会给你一个公网地址，格式类似：

```text
https://three-agent-card-room.onrender.com
```

把这个地址发给另外两位玩家即可。

## 玩家怎么进入

1. 三个人都打开同一个公网网址
2. 第一个人输入名字并创建房间
3. 页面会显示房间号
4. 另外两个人输入名字和房间号加入
5. 满 3 人后房主点击开始游戏

## 你现在不用再改的东西

- 不用再改端口
- 不用再改启动命令
- 不用额外安装第三方库

## 注意

- Render 免费版可能会休眠，第一次打开会稍慢几秒；
- 这个游戏状态保存在服务器内存里，服务重启后房间会清空；
- 所以它适合课堂作业、短局试玩，不适合长期保存牌局。
