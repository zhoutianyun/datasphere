# 三机对抗 - 联机房间版 - 游戏规则

## 游戏卡

```text
小组名称：__________
游戏名称：三机对抗 - 联机房间版

一句话目标：
3 个 Agent 在不同电脑上加入同一个房间，轮流出牌争抢资源，率先打开 2 个宝箱获胜。

怎样算胜利：
谁先拿到 2 分，谁立刻胜利。
如果 12 次总出牌结束还没人到 2 分，则按分数结算。

怎样算失败：
不是淘汰制，结算时没赢的玩家就是失败方。

什么时候必须停下来问人：
当“干扰牌”可攻击目标并列时，必须由人点击决定干扰谁。
如果最终分数和资源都完全并列，也必须由人点击决定胜者。

每个 Agent 可以做什么：
创建房间；
加入房间；
出自己手里的牌；
在页面要求时选择干扰目标或决胜目标。

行动后游戏会发生什么变化：
每次出牌后都会改变手牌、线索、钥匙、分数、剩余总回合或当前行动者。

玩家会马上看到什么反馈：
所有电脑页面会同步显示：
谁出了哪张牌；
当前谁领先；
每个玩家有多少线索、钥匙、分数；
还剩多少总回合；
是否已经胜利或需要人来决定。
```

## 房间玩法

1. 一位玩家创建房间；
2. 另外两位玩家输入房间号加入；
3. 满 3 人后自动开始游戏；
4. 按 `Agent 1 -> Agent 2 -> Agent 3 -> Agent 1 ...` 的顺序轮流出牌。

## 卡牌说明

- 调查牌：获得 1 条线索；
- 深度调查：直接获得 2 条线索；
- 钥匙牌：获得 1 把钥匙；
- 顺手牵钥：从钥匙最多的对手手里夺走 1 把钥匙；
- 防护牌：获得 1 层护盾，可抵消 1 次被干扰或被抢钥匙；
- 干扰牌：让领先对手失去 1 条线索；
- 万用牌：自动按当前局势补线索、补钥匙，或直接补足一次开箱；
- 开箱牌：若自己至少有 2 条线索和 1 把钥匙，则消耗它们并获得 1 分。

## 牌库随机性

- 每位玩家不再使用完全相同的固定 6 张牌；
- 现在每位玩家都会从更大的公共牌池里随机抽出 `7` 张组成自己的牌库；
- 所以不同对局里，拿到的功能牌组合会不同，随机性更强。

## 统一使用规则

- 3 个 Agent 使用同一张规则卡；
- 3 个 Agent 在各自电脑上操作，但共享同一局状态；
- 每次点击一张牌都算一次行动；
- 每次行动后所有页面都应同步出现变化；
- 如果触发“必须停下来问人”，对局暂停等待点击。

- [x] 目标和胜负条件一眼能看懂
- [x] 每次行动都会改变至少一个状态
- [x] 每次行动后都有清楚反馈
- [x] 有竞争结果和必须问人的情况
- [x] 3 个 Agent 用同一张规则卡
- [x] 可以在其他人的电脑上打开并加入同一局
