# 自动出牌策略说明

这份联机版现在额外带了一个更像“会思考的 Agent”的自动对局脚本：

`游戏文件/auto_play_room.py`

## 它和之前的区别

之前的自动出牌更像简单贪心：

- 能开箱就开；
- 不然就优先补线索或补钥匙；
- 很少考虑对手威胁。

现在这版会额外判断：

- 自己这回合能不能直接得分；
- 对手是不是已经接近开箱；
- 干扰牌该不该现在交；
- 偷钥匙是不是比补资源更值；
- 护盾是不是应该提前挂上；
- 万用牌是该攒着终结，还是先拿来补资源。

## 新策略的大致优先级

1. 如果这回合能直接得分，优先得分；
2. 如果对手下回合可能得分，优先干扰或偷钥匙；
3. 如果自己快成型，优先补足缺的那一项；
4. 如果局势危险且自己没护盾，优先上护盾；
5. 万用牌会按局势动态使用，不再固定乱交。

## 怎么运行

先确保 `server.py` 正在运行，然后执行：

```powershell
python ".\游戏文件\auto_play_room.py"
```

它会自动：

1. 创建房间；
2. 加入 3 个策略 Agent；
3. 满 3 人自动开局；
4. 自动把整局打完；
5. 最后输出完整结果和出牌历史。
